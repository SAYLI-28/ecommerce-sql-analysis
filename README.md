# Ecommerce Sales Analysis using SQL Server

## Project Overview

This project analyzes an ecommerce dataset using SQL Server to uncover business insights related to customer behavior, payment trends, revenue growth, and regional performance.

The project demonstrates:

* SQL Joins
* Aggregations
* GROUP BY
* CTEs
* Window Functions
* Date Conversion Handling
* Running Revenue Analysis

---

## Dataset Tables

* orders
* customers
* payments

---

## Key Analysis Performed

### Revenue Analysis

* Total revenue calculation
* Revenue by payment type
* Monthly revenue trends

### Customer Analysis

* Top spending customers
* Customer ranking using window functions

### Regional Analysis

* Orders by state
* Regional order concentration

### Advanced SQL Concepts

* TRY_CONVERT for date handling
* CTEs
* Running totals using window functions

---

## Key Business Insights

* Credit cards generated the highest revenue contribution.
* SP had the highest order concentration.
* Revenue fluctuated across months.
* High-value customers contributed disproportionately high revenue.
